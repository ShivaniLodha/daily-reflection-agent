# Design Rationale

This tree was built using three psychological axes:

1. Locus of Control
2. Contribution vs Entitlement
3. Radius of Concern

Design choices:
- Fixed-option questions
- Deterministic branching
- Signal based summaries

Guardrails:
- No runtime LLM use
- No hallucinated responses
- Static rule-driven routing

With more time I would add more nuanced branches.
